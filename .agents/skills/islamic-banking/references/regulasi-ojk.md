# Regulasi OJK & Kerangka Hukum Perbankan Syariah Indonesia

---

## 1. HIERARKI REGULASI

```
1. Undang-Undang
   - UU No. 21 Tahun 2008 tentang Perbankan Syariah
   - UU No. 7 Tahun 1992 jo. UU No. 10 Tahun 1998 tentang Perbankan
   - UU No. 21 Tahun 2011 tentang OJK

2. Peraturan Pemerintah / Perpres

3. Peraturan Bank Indonesia (PBI) — berlaku sampai beralih ke OJK
   - PBI No. 11/3/PBI/2009 tentang BUS
   - PBI No. 11/10/PBI/2009 tentang UUS

4. Peraturan OJK (POJK)
   - Bersifat mengikat secara hukum
   - Menggantikan PBI untuk pengawasan bank

5. Surat Edaran OJK (SE OJK)
   - Petunjuk teknis pelaksanaan POJK
   - Berisi formulir, tata cara, template pelaporan

6. Fatwa DSN-MUI
   - Landasan syariah produk
   - Wajib menjadi acuan bank syariah (bukan regulasi formal, tapi diadopsi OJK)
```

## 2. POJK UTAMA PERBANKAN SYARIAH

### Kelembagaan & Perizinan
```
POJK No. 64/POJK.03/2016 — Prinsip Kehati-hatian dalam Kegiatan Penyertaan Modal
POJK No. 6/POJK.03/2016  — Kegiatan Usaha dan Jaringan Kantor Berdasarkan Modal Inti
POJK No. 35/POJK.03/2023 — Perubahan Kegiatan Usaha UUS menjadi BUS (Spin-off)
```

### Permodalan & Kehati-hatian
```
POJK No. 11/POJK.03/2016 — Kewajiban Penyediaan Modal Minimum (KPMM) BUS
  → CAR minimum, ATMR, komponen modal Tier 1 & Tier 2
  
POJK No. 34/POJK.03/2016 — Prinsip Kehati-hatian dalam Penyaluran Pembiayaan

POJK No. 37/POJK.03/2019 — Perubahan POJK KPMM (penyesuaian Basel III)
  → Buffer Konservasi Modal 2.5%
  → Buffer Countercyclical
  → SIFI Buffer (D-SIB)
```

### Kualitas Aset & CKPN
```
POJK No. 40/POJK.03/2019 — Penilaian Kualitas Aset BUS dan UUS
  → Definisi kolektibilitas (Kol 1-5)
  → Kriteria restrukturisasi pembiayaan
  → Persyaratan agunan
  → BMPK (Batas Maksimum Pemberian Kredit/Pembiayaan)

POJK No. 11/POJK.03/2020 — Stimulus Perekonomian Nasional (COVID-19)
  → Relaksasi kolektibilitas pembiayaan restrukturisasi
  → Berlaku s/d Maret 2023 (diperpanjang beberapa kali)

SE OJK No. 19/SEOJK.03/2019 — Perubahan Penilaian Kualitas Aset
  → Kriteria teknis penentuan kolektibilitas
```

### Tingkat Kesehatan Bank (TKS)
```
POJK No. 8/POJK.03/2014 — Penilaian Tingkat Kesehatan BUS dan UUS
  → Kerangka RGEC (Risk, GCG, Earnings, Capital)
  → Kewajiban penilaian self-assessment per semester
  → Kewajiban pelaporan ke OJK

SE OJK No. 10/SEOJK.03/2014 — TKS BUS dan UUS
  → Panduan teknis penilaian per faktor
  → Formulir penilaian
  → Threshold rasio per peringkat
```

### Manajemen Risiko
```
POJK No. 65/POJK.03/2016 — Penerapan Manajemen Risiko bagi BUS dan UUS
  → 10 jenis risiko (termasuk Risiko Imbal Hasil & Risiko Investasi)
  → Kerangka Risk Appetite Statement (RAS)
  → Three Lines of Defense

SE OJK No. 13/SEOJK.03/2015 — Penerapan Manajemen Risiko BUS
  → Kebijakan, prosedur, dan limit risiko
  → Stress testing dan skenario analisis

POJK No. 18/POJK.03/2016 — Penerapan Manajemen Risiko dan Tata Kelola TI
POJK No. 21/POJK.03/2023 — Penyelenggaraan Sistem IT Bank Umum
```

### GCG (Good Corporate Governance)
```
POJK No. 55/POJK.03/2016 — Penerapan Tata Kelola Bagi BUS
  → Komposisi Dewan Komisaris & Direksi
  → Komite-komite (Audit, Risiko, Remunerasi, Nominasi)
  → Dewan Pengawas Syariah (DPS)
  → Larangan rangkap jabatan

POJK No. 33/POJK.04/2014 — Direksi dan Dewan Komisaris Emiten/Perusahaan Publik
  (berlaku jika bank syariah sudah tbk/listed)
```

### Pelaporan
```
SE OJK No. 43/SEOJK.03/2016 — LK Berkala BUS & UUS
  → Format Neraca, LR, CAR, NPF untuk publikasi
  → Jadwal pelaporan

SE OJK No. 14/SEOJK.03/2017 — LK Triwulanan BUS
  → Format laporan triwulanan untuk publikasi media

POJK No. 12/POJK.03/2019 — Pelaporan Bank Umum melalui Sistem Pelaporan OJK
  → Sistem SiPEBI (Sistem Pelaporan BI diganti)
  → Template XML pelaporan

POJK No. 4/POJK.03/2023 — Perubahan Laporan Bank Umum
```

### Produk Syariah
```
POJK No. 24/POJK.03/2015 — Produk dan Aktivitas BUS & UUS
  → Daftar produk yang diizinkan
  → Prosedur perizinan produk baru
  → Penerapan fatwa DSN-MUI

POJK No. 01/POJK.07/2013 — Perlindungan Konsumen Sektor Jasa Keuangan
POJK No. 6/POJK.07/2022 — Perlindungan Konsumen (update)
  → Transparansi produk, penyelesaian pengaduan
```

### BMPK (Batas Maksimum Pemberian Kredit/Pembiayaan)
```
POJK No. 32/POJK.03/2018 — BMPK & BMPD BUS
  → Pihak Terkait (related parties): max 10% Modal
  → Pihak Tidak Terkait Grup: max 25% Modal
  → Seluruh Pihak Terkait: max 20% Modal
  → Batas Maksimum Penyertaan Dana (BMPD)
```

## 4. SISTEM PELAPORAN KE OJK (SiPEBI / LSMK)

```
Platform: SiPEBI (Sistem Informasi Pelaporan Bank Indonesia — digantikan oleh
          SiPEBI generasi baru + sistem OJK)

Laporan Bulanan:
  - Laporan Keuangan Publikasi Bulanan (Neraca + LR + Off Balance Sheet)
  - Laporan Keuangan dalam format XBRL
  - Laporan GWM
  - Laporan PDN

Laporan Triwulanan:
  - Laporan Keuangan Triwulanan Publikasi
  - Laporan Profil Risiko
  - Laporan GCG (self-assessment)

Laporan Semesteran:
  - Laporan TKS (RGEC Assessment)
  - Laporan Penilaian GCG

Laporan Tahunan:
  - Laporan Keuangan Auditan
  - Laporan Tahunan (Annual Report)
  - Laporan Pelaksanaan GCG
  - Rencana Bisnis Bank (RBB) untuk tahun berikutnya

Sanksi Keterlambatan Pelaporan:
  - Rp 1.000.000/hari kerja pertama
  - Rp 2.000.000/hari kerja berikutnya
  - Maksimum Rp 50.000.000 per laporan
```

## 5. PERPAJAKAN KHUSUS BANK SYARIAH

```
PPh Bagi Hasil Deposito Mudharabah:
  Tarif: 20% (sama dengan bunga deposito konvensional)
  Dasar: PP No. 131 Tahun 2000
  
PPh Margin Murabahah:
  Diperlakukan sama dengan bunga kredit konvensional
  Penghasilan bank: objek PPh Badan tarif normal

PPN Produk Syariah:
  PMK No. 40/PMK.010/2010: IMBT — penyerahan hak atas tanah/bangunan tidak
  dikenakan PPN ganda (hanya sekali di awal)
  
BPHTB & PPh Pasal 25/29:
  Transaksi IMBT / MMQ: PMK mengatur hanya satu kali BPHTB
  (saat penyerahan akhir ke nasabah, bukan saat akad awal)

Zakat:
  PP No. 60 Tahun 2010: Zakat yang dibayarkan dapat menjadi pengurang
  penghasilan bruto (bukan pengurang pajak/tax credit)
```
